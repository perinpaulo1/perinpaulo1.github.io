Data Project - Exploring data from E-commerce

One of the most valuable data is the data from e-commerce, this is the reason that I've decided to choose a public dataset from Kaggle with 100k orders from Brazilian e-commerce. To explore the data set I've chosen to answer 3 questions that may help the seller and the customers. The 1st question was: Who are the customers from a specific vendor? the answer may help the vendor to build a more solid market strategy. The other 2 questions are related to how to help to meet the needs of a client, by showing a map with the closest vendor or showing the best price of a specific category of product.

    • To use it was required to apply ETL process to transform/clean the data and avoid mistakes. Once the data was ready to use, it was necessary to choose a system of coordinates that might bring the most accurate locations and distances. After some research, I've decided to follow the geodetic coordinate, since the geodetic takes the globe curves on its calculus, and for a more accurate location the geodesic coordinates system for Latin America, SIRGAS2000.Pandas is undeniably one of the most famous Python packages for data manipulation, for geo-location I used the geopandas a really useful package to work with geospatial data.

    To the 1st question, there is the class mycustomer using the seller_id, which creates an object with the method to create a data frame with the customers of a vendor. The data frame will provide the locations, the freight value, and how much that customer had spent. The class best_buy is answering the remaining question, with the customer id, the product category, the number of records, and the parameters of fast to get the closest vendor or best_price that will return the best prices near the customers. The object created by the best_buy class has two methods one to create a data frame with the information of the type of buy and another method that will plot a map with the customer and vendor's location.
